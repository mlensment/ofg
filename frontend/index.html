What is it?

One File Gallery is a neat little tool to share pictures with your friends. No configuration needed whatsoever.

How to get it?

Just download it right here or follow the link to Github.

How to install it?

First, make sure your server supports PHP 5.
Extract the file you just downloaded and upload the extracted directory to your server.
All done!
Now you just need to upload your pictures. The folder structure looks like that: 

galleries/gallery1
galleries/gallery2

The directory name determines the title that will be displayed on the gallery’s homepage. Underscores will translate into spaces.

my_gallery -> my gallery

Pick your favourite browser and navigate to your new gallery (eg. http://yourdomain.com/myGallery/)

How it’s made?

One File Gallery is programmed in PHP, Javascript, HTML5 and CSS3.
